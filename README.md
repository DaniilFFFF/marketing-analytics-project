# 📊 Marketing Analytics Project

## Про проєкт

Цей проєкт створюється як повноцінна симуляція роботи Junior Data Analyst. Його мета — пройти весь цикл роботи з даними: від ознайомлення з датасетом і побудови моделі даних до створення аналітичних дашбордів та бізнес-висновків.

Проєкт виконується поетапно з використанням сучасного стеку інструментів, які найчастіше використовуються в українських та міжнародних компаніях.

---

## Мета проєкту

Побудувати повний аналітичний пайплайн для аналізу маркетингової ефективності інтернет-магазину та відповісти на ключові бізнес-питання за допомогою сучасних інструментів аналітики.

---

## Бізнес-задача

Компанія хоче зрозуміти:

- які маркетингові канали працюють найефективніше;
- які товари приносять найбільшу виручку;
- як змінюються продажі в часі;
- які користувачі приносять найбільшу цінність бізнесу;
- які фактори впливають на конверсію.
